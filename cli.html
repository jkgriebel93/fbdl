Command Line Interface

This project exposes a command-line interface under the fbdl entry point.

fbdl

fbdl [OPTIONS] COMMAND [ARGS]...

convert-format

Convert mkv files stored in DIRECTORY to mp4 files. Other formats will be added eventually.

DIRECTORY is the directory fbdl will search for mkv files to convert.

fbdl convert-format [OPTIONS] DIRECTORY

Options

--orig-format <orig_format>

fbdl will attempt to convert all videos with the file extension provided here.

--new-format <new_format>

The desired output file type.

--pretend

If passed, don’t perform actual updates. Preview only.

--delete

If passed, remove the mkv files after conversion.

Arguments

DIRECTORY

Required argument

download-list

Download a list of URLS provided via INPUT_FILE and store the files in OUTPUT_DIRECTORY.

INPUT_FILE is the path to a text file where video URLs are listed, one per line. OUTPUT_DIRECTORY is the path to store the downloaded videos in.

fbdl download-list [OPTIONS] INPUT_FILE OUTPUT_DIRECTORY

Options

--cookie_file <cookie_file>

If authentication is required, you can provide a cookies file via this flag. The file must follow the Netscape format.

Arguments

INPUT_FILE

Required argument

OUTPUT_DIRECTORY

Required argument

nfl-games

Download NFL game replays of the specified SEASON and WEEK

SEASON Is the year (2009 or later) for which to download replays. WEEK The season week number to download replays for.

fbdl nfl-games [OPTIONS] SEASON WEEK

Options

--team <team>

Restrict downloads to a single team by providing the team’s three letter abbreviation here. If blank, all are fetched.

--replay-type <replay_type>

Specify which replay types to download. If blank, the full game is downloaded.

Options:

full_game | all_22 | condensed_game | full_game_alternative

--start-ep <start_ep>

Where to pick up episode numbering from.

--raw-cookies <raw_cookies>

A txt file containing cookies needed for NFL API authentication.

Arguments

SEASON

Required argument

WEEK

Required argument

nfl-show

Download episodes from shows available on NFL Plus.

INPUT_FILE is a JSON file containing the URL leaf nfl.com uses for each episode.

fbdl nfl-show [OPTIONS] INPUT_FILE

Options

--output-directory <output_directory>

The path to store the downloaded videos in.

--cookies <cookies>

A .txt file containing NFL authentication cookies following the Netscape format.

Arguments

INPUT_FILE

Required argument

rename-series

A one-off command used to change file format names from SEE to <Series Name> (YYYY) - sSeEE - <episode_name>

SERIES_NAME is the name of the TV series

fbdl rename-series [OPTIONS] SERIES_NAME

Options

--pretend

If passed, don’t perform actual updates. Preview only.

--release-year <release_year>

The year that the show first aired.

--replace

If passed, overwrite any file that already exists with the new name.

Arguments

SERIES_NAME

Required argument

update-metadata

A largely one-off command to update embedded metadata for already downloaded NFL games based on the file’s name.

DIRECTORY_PATH is the directory fbdl will search for mp4 files.

fbdl update-metadata [OPTIONS] DIRECTORY_PATH

Options

--pretend

Don’t perform any updates; preview only.

--verbose

Enable extra logging.

Arguments

DIRECTORY_PATH

Required argument
